🚀 A full-stack AI chatbot application that allows users to create, manage, and continue conversations with an AI model, similar to ChatGPT.

## 🚀 Features
-💬 Create new chats and continue previous conversations
🧠 AI-generated responses using Gemini API
🗂️ Multiple chat threads management
💾 Persistent chat storage in database
🎨 Clean and responsive UI


  ## 🛠 Tech Stack
- Backend: Node.js, Express.js
- Database: MongoDB, Mongoose
- Frontend: React.js, CSS
- Authentication: Passport.js
- AI Integration: Gemini API
